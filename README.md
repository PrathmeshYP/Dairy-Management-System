# Dairy Management System 

This project is a Python-based Dairy Management System developed as a SAM Mini Project.
It is designed to manage basic dairy operations such as milk receipt entry, report generation, and database connectivity in a simple and efficient way.
The project is suitable for academic and learning purposes and follows a modular programming approach.

# Features

* Milk receipt entry functionality
* Simple user interface for managing milk records
* Milk report generation
* Database connection testing
* Modular and easy-to-understand code structure

# How to Run the Project

Clone the repository
git clone (https://github.com/PrathmeshYP/Dairy-Management-system.git)

Open the project folder
cd Sam-mini

Run the main program
python dairy_management.py

# Database Connection Test

To test the database connection, run:
python test_connection.py

# Milk Reports

To generate milk reports, run:
python milk_reports.py

# Requirements
